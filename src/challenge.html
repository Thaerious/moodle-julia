Challege 1

Create a function whcih acceps an array of integers. This function returns a new array with all the values from the initial array in reverse order.

Example 1: anArray = [1, 7, 3, 5, 2] r = reverseArray(anArray) println(resize!(anArray, r)) output> [2, 5, 3, 7, 1]

Challege 2

Given an array of integers, remove all duplicate values and return the new array size. The values in the new array must remain in order. The 'resize!' function will be used to truncate the array, so extraneous values do not matter. The array should be manipulated in place, such that no external data structure is needed beyond a few simple varaibles.

Example 1: anArray = [1, 2, 2, 3, 4, 5, 5, 5, 6] r = removeDuplicates(anArray) println(resize!(anArray, r)) output> [1, 2, 3, 4, 5, 6]

Example 2: anArray = [1, 1, 1, 4, 5, 5, 5, 6, 9, 10] r = removeDuplicates(anArray) println(resize!(anArray, r)) output> [1, 4, 5, 6, 9, 10]

Challege 3

Given an unsorted array of integers. Implement the 'Bubble Sort' algorithm to return a sorted array. Sort in-place, do not create a new array.

Example 1: anArray = [1, 3, 2] bubbleSort(anArray) println(anArray) output> [1, 2, 3]

Example 2: anArray = [1, 4, 4, 2, 8, 10, 7] bubbleSort(anArray) println(anArray) output> [1, 2, 4, 4, 7, 8, 10]
