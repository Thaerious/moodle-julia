Selecting a Subset
Here we have two conditions, cars.mpg .>= 20 and cars.hp .>= 100. Each returns an array of boolean values. When the .& operator combines them.
cars[(cars.mpg .>= 20) .& (cars.hp .>= 100),:]
Combining conditions when selecting rows and columns.
We can use the select() function to select a subset of the DataFrame.
select(cars,Not([:mpg,:cyl]))
