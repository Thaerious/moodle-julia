Installing Packages

To use official Julia module you download and install the package containing them.  You can browse the available packages here: https://julialang.org/packages/.  To download and install a package, press ']' from within the REPL. Press [backspace] or [ctrl]-C to exit the package manager. Use the 'add' command to add a package to the package manager. Use the 'status' command sto see what packages you have installed. The help command will list all available package manager commands.  If you want to see extra details on a command use 'help [command]'.

julia> ]
    (@v1.4) pkg> add Calculus
    (@v1.4) pkg> status
    Status `C:\Users\edwar\.julia\environments\v1.4\Project.toml`
    [49dc2e85] Calculus v0.5.1
    [587475ba] Flux v0.10.4
    [4076af6c] JuMP v0.21.2
    (@v1.4) pkg> help
    .
    .
    .
Back in the REPL (or source file) we use the keyword 'using' to make the package available to the current session.  After we have declared 'using', we can view all symbols in the package with the 'names' function.  Although for any real information it is best to visit the documentation page (https://juliahub.com/ui/Packages/Calculus/G3wEN/0.5.1). 
julia> using Calculus

julia> Calculus.differentiate("x^2", :x)
:(2 * 1 * x ^ (2 - 1))
